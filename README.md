# Sudoku-Project
Name: Aya Sophia Cheng

Description: This project works to generate a Sudoku Board with random values from 1-9 that follow the column, row, and box rules. It then removes 10 of the values for the player to solve for (the answers are still printed first though just to show the non-extra credit part of this project..) Then it creates a game loop where the player can enter the coordinate they want to answer for and then enter a value.

How to Run the Program: When you run the program the only thing the player has to do is input values when prompted to try and solve for the Sudoku board. They also have the option to quit if they wish to.

How the board is generated: The board is generated by first assigning a sequence of values 1-9 to the top-left 3x3 box, the center box, and the bottom right box. This is done because their rows and columns do not interfere with each others. Then we are directed to another method called "fillRestOfBoard" where it uses recursion to finalize and fill in the rest of the board that does not already have a value.
